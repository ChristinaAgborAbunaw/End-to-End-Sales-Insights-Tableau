# 📊 End-to-End Sales Insights Project using Tableau

## Overview

This project explores sales performance using the Superstore Sales dataset to uncover trends, evaluate profitability, and generate actionable business insights.

The project follows an end-to-end analytics workflow including data preparation, analysis, visualization, and reporting.

---

## Business Objective

Analyze sales performance and identify opportunities to improve profitability, understand customer behavior, and support data-driven business decisions.

---

## Dataset

**Dataset:** Superstore Sales Dataset

**Source:** Tableau Public Sample Dataset

### Tables Included

* Orders
* People
* Returns

### Records

Approximately 9,900+ sales transactions

---

## Project Goals

* Analyze overall sales performance
* Identify top-performing categories and regions
* Evaluate profitability
* Understand customer purchasing behavior
* Investigate product returns
* Build an interactive Tableau dashboard

## Business Questions

1. Which category generated the highest sales?
2. Which region generated the highest profit?
3. Which customer segment generated the highest revenue?
4. What sales trends exist over time?
5. Which products underperformed?
6. How do returned orders affect profitability?
